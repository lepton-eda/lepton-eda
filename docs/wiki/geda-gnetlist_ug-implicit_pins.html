Implicit pins

As an alternative to using visible pins drawn in a symbol, gnetlist supports defining connections between named nets and pins using the ”net=” attribute. This is commonly used with slotted components, to avoid either having to add power pins to each slot or having to place an separate “power connections” component into the schematic.

For more information on using the ”net=” attribute, see the net= attribute HOWTO.
