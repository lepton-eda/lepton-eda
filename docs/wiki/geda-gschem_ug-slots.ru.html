Секции компонентов и их назначение

В Руководстве по созданию символов приводится описание того, как создавать символы для многосекционных компонентов.

В некоторых интегральных схемах в одном корпусе содержится по несколько одинаковых взаимозаменяемых устройств. В gschem и gnetlist для поддержки работы с ними используются компоненты с секциями (многосекционные компоненты).

Для использования многосекционного компонента создайте в схеме несколько копий этого компонента и задайте им всем одинаковое позиционное обозначение в атрибуте refdes. Затем выделите один из этих компонентов и откройте окно “Правка атрибутов” [“Edit Attributes”] с помощью Правка→Правка… [Edit→Edit…]. Если вы установите флажок “Показать унаследованные атрибуты” [“Show inherited attributes”], то должны будете увидеть атрибут numslots, в котором задаётся количество секций в корпусе.

Для установки соответствия копии компонента номеру секции в корпусе эту копию нужно выделить и открыть для неё окно “Правка номера секции” [“Edit Slot Number”] с помощью Правка→Секция… [Edit→Slot…]. При изменении номеров секций автоматически обновляются и номера выводов компонента на экране.
