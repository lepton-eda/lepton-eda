НАЗВАНИЕ
       gsymcheck - программа проверки символов gEDA/gaf

СИНТАКСИС
       gnetlist [-h] [-v] [-q] symbol1 [... symbolN]

ОПИСАНИЕ
       gsymcheck - это программа проверки символов для gEDA. Сюда нужно
       добавить перечень выполняемых для символов проверок.


ПАРАМЕТРЫ
       gsymcheck принимает следующие ключи:

       -q      Безмолвный режим. В этом режиме отключаются все
               предупреждения/ замечания/сообщения. (необязательный)

       -v      Подробный режим 1. В этом режиме выводятся все ошибки.
               (необязательный)

       -vv     Подробный режим 2.	В этом режиме выводятся все ошибки и
               предупреждения. (необязательный)

       -vvv    Подробный режим 3.	В этом режиме выводятся все ошибки,
               предупреждения и информационные сообщения. (необязательный)

       -h      Сводка по использованию / справка по gsymcheck

       symbol1 [... symbolN]
               Должен быть указан по крайней мере один файл символа. Если
               указано несколько символов, они последовательно читаются и
               проверяются.  Важно, чтобы файлы символов задавались после
               задания всех ключей (то есть в конце командной строки).
