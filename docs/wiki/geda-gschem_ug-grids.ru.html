Сетки

Для размещения объектов схемы или символа в gschem используется сетка. В любое время активны две сетки: сетка привязки и отображаемая сетка. Текущие настройки сеток отображаются в строке состояния gschem в виде “Сетка(100,100)” [“Grid(100,100)”], где первое число относится к сетке привязки, а второе — к отображаемой сетке.

Сетка привязки

Сетка привязки используется для операций редактирования, она позволяет позиционировать объекты таким образом, чтобы обеспечить их беспроблемное подключение друг к другу. Размер сетки можно увеличить или уменьшить, выбрав в меню соответственно Настройки→Увеличить шаг сетки [Options→Scale up grid spacing] или Настройки→Уменьшить шаг сетки [Options→Scale down grid spacing]. С помощью Настройки→Шаг сетки привязки… [Options→Snap grid spacing…] можно задать нужное нестандартное значение шага.

Текущий режим привязки можно переключать с помощью Настройки→Переключить привязку [Options→Toggle Snap On/Off]. Имеется три режима привязки:

  1. В обычном режиме перемещения всех объектов производятся на расстояние, кратное шагу сетки привязки, но объекты, по сетке не выровненные, принудительно выставляться по ней не будут.
  2. В режиме перепривязки в строке состояния отображается “Режим перепривязки” [“Resnap Active”]. В этом режиме те объекты, что не выровнены по сетке, будут снова принудительно по ней выставляться. Этот режим может быть полезен в том случае, когда у вас есть объект, выровненный по сетке другого размера, нежели тот, что используется в данный момент.
  3. В режиме работы без привязки в строке состояния отображается “Привязка отключена” [“Snap Off”]. В этом режиме объекты вообще не прилипают к сетке.

Будьте осторожны при работе с отключенной сеткой: к выводам, соединениям и компонентам, немного смещённым относительно сетки, порой бывает совершенно невозможно что-либо подключить.

Отображаемая сетка

Отображаемая сетка имеет минимальный приемлемый для отображения в текущем масштабе размер, кратный шагу сетки привязки. Вид сетки переключается с помощью Настройки→Стиль сетки [Options→Cycle grid styles].
