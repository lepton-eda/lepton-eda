Аргументы командной строки

При запуске gnetlist может использоваться несколько необязательных аргументов командной строки. Их перечень можно узнать, запустив gnetlist -h. Перечень всех ключей командной строки программы можно посмотреть в её справочной странице (man gnetlist).

Вывод сообщений и диагностика

-q Без вывода информации. Отключение всех предупреждений/замечаний/сообщений.
-v, --verbose Подробный вывод. Вывод дополнительной диагностической информации.
-h, --help Вывод справки.
-V, --version Вывод информации о версии gnetlist.
--list-backends Вывод информации об установленных драйверах создания списков соединений.

Обработка списка соединений

-o FILE Записать сгенерированный список соединений в FILE.
-g BACKEND Указать, какой драйвер использовать для генерации данных списка соединений.
-O STRING Передать опцию STRING драйверу списка соединений.
-i Войти в интерактивный режим Scheme REPL вместо запуска драйвера.
-- Рассматривать все оставшиеся аргументы как имена файлов схем или символов. Используйте это, если у вас есть имена файлов схем, начинающиеся с «-».

Использование расширений на Scheme

-L КАТАЛОГ Добавление КАТАЛОГа к началу списка каталогов, в которых производится поиск файлов Scheme.
-c ВЫРАЖЕНИЕ Задание выражения на Scheme, которое должно быть вычислено при запуске.
-l ФАЙЛ Задание скрипта на Scheme, который должен быть выполнен перед загрузкой драйвера.
-m ФАЙЛ Задание скрипта на Scheme, который должен быть выполнен после загрузки драйвера.
