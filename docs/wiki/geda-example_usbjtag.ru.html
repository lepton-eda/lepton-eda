Пример интерфейса JTAG на базе USB в каталоге:

где ${prefix} — установочный префикс для gEDA в вашей системе (обычно /usr или /usr/local). 

gTAG - устройство сопряжения USB и JTAG
=======================================

Это файл README для схемы gTAG.

Краткое описание
----------------
gTAG -- это устройство сопряжения для USB-порта вашего компьютера,
предназначенное для обмена информации со схемами, поддерживающими JTAG.

Авторское право
---------------
Владелец авторского права (C) на эти схемы -- Stefan Petersen
(spe@stacken.kth.se), схемы выпущены под лицензией GPL (см. прикреплённый файл
COPYING). GPL написана главным образом для программного обеспечения, то есть
для интеллектуальной собственности в электронном виде. Определение схемы
gTAG как интеллектуальной собственности в электронном виде позволяет
выпускать её под лицензией GPL.

Как?
----
Эти схемы созданы с помощью gschem, а списки соединений сгенерированы затем с
помощью gnetlist. И gschem, и gnetlist -- это части gEDA
(http://www.geda-project.org).

Файлы
-----
Дистрибутив схем gTAG должен состоять из следующих файлов:
* README - данный файл;
* COPYING - уведомление об авторском праве GPL;
* crdist.sh - скрипт shell, используемый для создания tar.gz;
* ChangeLog - попытка нацарапать, что изменилось между версиями;
* Четыре символа компонентов:
  - 7414-1.sym - оригинальный символ gschem 7414 без атрибутов питания;
  - max882.sym - преобразователь 5 В в 3.3 В;
  - cy7c64603-52nc.sym - USB-микроконтроллер от Cypress;
  - sn75240pw.sym - ограничитель перенапряжений для USB;
  - copyleft.sym - символ с текстом GPL для всех схем.
* Пять схем с четырьмя символами:
  - gTAG.sch.sch - основная схема для gTAG;
  - gTAG-jtagio.[sym/sch];
  - gTAG-ucont.[sym/sch];
  - gTAG-consio.[sym/sch];
  - gTAG-psu.[sym/sch];
* rc-файлы gEDA для этого проекта:
  - commonrc   - содержит общие определения и для gschem, и для gnetlist;
  - gnetlistrc - специфичен для gnetlist, вызывает commonrc;
  - gschemrc   - специфичен для gschem, вызывает commonrc;
