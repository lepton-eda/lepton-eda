Запуск gschem

Вам нужна справка по установке gEDA?

gschem можно запустить прямо из меню приложений, поищите пункт “gEDA Schematic Editor”, который на большинстве Linux-систем можно найти в подменю “Engineering” или “Electronics”. Программу также можно запустить из менеджера файлов, открыв в нём файл схемы (с расширением .sch).

Кроме того, gschem можно запустить и из командной строки, набрав команду gschem. Чтобы открыть новую схему, без названия, команду можно задать без аргументов: 

gschem

Можно открыть и несколько файлов схем или символов, указав в команде их имена: 

gschem mydesign.sch symbols/resistor-1.sym

Если какой-то из заданных файлов ещё не существует, gschem создаст новый файл с указанным именем.

Программа gschem принимает несколько дополнительных аргументов, которые можно задать в строке команды для изменения её поведения. Они подробно описаны в справочной странице gschem(1) (которую можно посмотреть, запустив man gschem).
